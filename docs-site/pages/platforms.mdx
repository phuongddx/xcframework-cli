## iOS

Build for iOS devices and simulators.

### iOS Device (`ios`)

```yaml
frameworks:
  - name: MySDK
    platforms: [ios]
    deployment_targets:
      ios: "14.0"
    architectures:
      ios: [arm64]
```

**SDK:** `iphoneos`
**Destination:** `generic/platform=iOS`
**Architectures:** `arm64`
**Minimum Target:** iOS 14.0

### iOS Simulator (`ios-simulator`)

```yaml
frameworks:
  - name: MySDK
    platforms: [ios-simulator]
    deployment_targets:
      ios-simulator: "14.0"
    architectures:
      ios-simulator: [arm64, x86_64]
```

**SDK:** `iphonesimulator`
**Destination:** `generic/platform=iOS Simulator`
**Architectures:** `arm64`, `x86_64`
**Minimum Target:** iOS 14.0

### Combined iOS Build

Build for both device and simulator:

```yaml
frameworks:
  - name: MySDK
    platforms: [ios, ios-simulator]
    deployment_targets:
      ios: "14.0"
```

This creates a universal XCFramework with:
- `ios-arm64` slice for devices
- `ios-arm64-simulator` and `ios-x86_64-simulator` slices for simulators

## Multi-Platform Builds

Build for multiple platforms simultaneously:

### iOS + macOS

```yaml
frameworks:
  - name: UniversalSDK
    platforms: [ios, ios-simulator, macos]
    deployment_targets:
      ios: "14.0"
      macos: "11.0"
```

### All Platforms

```yaml
frameworks:
  - name: UniversalSDK
    platforms:
      - ios
      - ios-simulator
      - macos
      - tvos
      - tvos-simulator
    deployment_targets:
      ios: "14.0"
      macos: "11.0"
      tvos: "14.0"
```

## Platform-Specific Settings

### Custom Architectures

Override default architectures per platform:

```yaml
frameworks:
  - name: MySDK
    platforms: [ios, ios-simulator]
    architectures:
      ios: [arm64]
      ios-simulator: [arm64]  # Exclude x86_64
```

### Custom Deployment Targets

Set different minimum versions per platform:

```yaml
frameworks:
  - name: MySDK
    platforms: [ios, macos]
    deployment_targets:
      ios: "15.0"     # Require iOS 15+
      macos: "12.0"   # Require macOS 12+
```

## Build Settings by Platform

Apply platform-specific build settings:

```yaml
build:
  build_settings:
    # Global settings
    ENABLE_BITCODE: "NO"

    # Platform-specific (via conditionals)
    OTHER_SWIFT_FLAGS: "-no-verify-emitted-module-interface"
```

## Troubleshooting

### Architecture Mismatches

If you see architecture errors:

```yaml
# Exclude problematic architectures
build:
  build_settings:
    EXCLUDED_ARCHS: "x86_64"
```

### Deployment Target Issues

Ensure deployment targets are set:

```yaml
frameworks:
  - name: MySDK
    deployment_targets:
      ios: "14.0"
      macos: "11.0"
```

### Platform Not Available

Check Xcode SDK availability:

```bash
xcodebuild -showsdks
```
